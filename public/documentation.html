myFlix API

This is an API that gives users access to a movie database where they can search for movies and information about them such as, director, artwork, genre. etc. Users can also sign up, update their personal information and create a list of their favorite movies.

Business Logic URL HTTP Method Request body data format Response body data format
Returns a list of ALL movies to the user /movies GET None A JSON object holding data about all the movies, Example: { "Title": "The Wizard of Oz", "Director": { "Name": "Victor Fleming" }, "Genre": { "Name": "Fantasy", "Description": "The fantasy genre is defined by both circumstance and setting inside a fictional universe with an unrealistic set of natural laws." } }
Returns data about a single movie to the user /movies/:Title GET None A JSON object holding data about a single movie including (description, genre, director, image URL). Example: { "Title": "The Wizard of Oz", "Director": "Victor Fleming"}
Returns data about a movie genre, including name and description /movies/genre/:genreName GET None A JSON object holding data about a single movie genre
Returns data about a director by name /movies/directors/:directorName GET None A JSON object holding data about a single director by name, (including bio, birth year, death year) Example:{ "Name": "David Cronenberg" }
Allows new users to register /users POST A JSON object holding data about the user to add, structured like: { ID: Integer, Username: String, Password: String, Email: String, Birthday: Date } A JSON object holding data about the user that was added, structured like: { ID: Integer, Username: String, Password: String, Email: String, Birthday: Date }
Allows users to update their user info (username) /users/:userId PUT None A text message indicating that the username was updated
Allows users to add a movie to their favorites list /users/:userId/favoriteMovies/:movieTitle PUT None A text message that a movie has been added to favorites list
Allows users to remove a movie from their favorites list /users/:userId/favoriteMovies/:movieTitle DELETE None A text message that a movie has been removed from favorites
Allows existing users to deregister /users/:userId DELETE None A text message that the user email has been removed
