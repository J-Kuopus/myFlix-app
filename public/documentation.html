myFlix API

This is an API that gives users access to a movie database where they can search for movies and information about them such as, director, artwork, genre. etc. Users can also sign up, update their personal information and create a list of their favorite movies.

Business Logic URL HTTP Method Request body data format Response body data format
Returns a list of ALL movies to the user /movies GET None A JSON object holding data about all the movies

Allows new users to register /users/register POST A JSON object holding data about the user to add, structured like: { name: John Doe } A JSON object holding data about the user that was added
Allows users to update their user info (username) /users/[username] PUT None A text message indicating that the username was updated
Allows users to add a movie to their favorites list /users/favorites PUT None A text message that a movie has been added to favorites list
Allows users to remove a movie from their favorites list /users/favorites DELETE None A text message that a movie has been removed from favorites
Allows existing users to deregister /users DELETE None A text message that the user email has been removed
